БОТ ПО ПОИСКУ ЭЛЕКТРОГИТАР С САЙТА AVITO

Интеллектуальный Telegram-бот для поиска электрогитар с визуальным выбором цвета и удобным интерфейсом. Бот помогает пользователю поэтапно отфильтровать нужную модель и найти подходящие объявления по бюджету.

* Основные возможности
Многоступенчатый поиск: Бренд → Модель → Цвет → Бюджет.

Визуализация цветов: Бот генерирует превью цвета прямо в чате, используя HEX-коды из базы данных.

UX "Чистый чат": Интерфейс работает в режиме "одного сообщения" — бот редактирует или удаляет свои старые сообщения, чтобы не засорять историю чата.

Гибкий бюджет: Выбор цены через быстрые кнопки или ручной ввод числа.

Прокси-интеграция: Настроен для обхода блокировок через HTTP-прокси с авторизацией.

* Стек
Интерпретатор: Python 3.12 (рекомендуется 3.10+)

Библиотека бота: aiogram 3.x (асинхронная работа)

База данных: SQLite3

Работа с изображениями: Pillow (для генерации квадратов с цветом)

Сетевые запросы: aiohttp с поддержкой прокси

* Инструкция по установке и запуску
1. Подготовка окружения
Рекомендуется создать виртуальное окружение:

python -m venv .venv
source .venv/Scripts/activate

2. Установка зависимостей
Для работы проекта необходимы библиотеки для работы с Telegram API и картинками:

pip install aiogram aiohttp pillow

3. Настройка конфигурации
В файле main.py уже прописаны текущие настройки доступа.

API_TOKEN: Токен бота от @BotFather.

* Структура проекта

main.py — Ядро бота. Содержит логику переключения экранов, обработку кнопок и систему удаления сообщений.

database.py — Слой работы с данными. Здесь находится функция init_db (создание таблиц) и SQL-запросы для поиска.

utils.py — Вспомогательный инструмент для генерации картинок-превью цвета гитары.

guitars.db — Файл базы данных SQLite.


* Важные примечания
База данных: Если вы меняете структуру колонок в database.py, удалите старый файл guitars.db, чтобы бот пересоздал его корректно.
